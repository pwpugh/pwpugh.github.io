JavaScript Variables

Variables store data values. You can use let, const, or var.

// Declaring variables
let name = "Student";
const age = 20;
var city = "Greensboro";

console.log(name); // Output: Student

Best Practice: Use const for values that won't change, and let for values that will.

JavaScript Functions

Functions are reusable blocks of code that perform specific tasks.

// Function declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Calling the function
console.log(greet("CodeAggie")); 
// Output: Hello, CodeAggie!

Tip: Functions help organize your code and make it reusable!

JavaScript Loops

Loops repeat code multiple times. The for loop is most common.

// For loop example
for (let i = 1; i <= 5; i++) {
    console.log("Count: " + i);
}

// Output:
// Count: 1
// Count: 2
// Count: 3
// Count: 4
// Count: 5

Remember: Loops save you from writing repetitive code!
