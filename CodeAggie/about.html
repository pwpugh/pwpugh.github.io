About CodeAggie

Our Mission

CodeAggie is an educational platform designed to help NC A&T students and beginners learn JavaScript fundamentals through interactive quizzes and practical examples. We believe learning to code should be accessible, engaging, and fun!

What We Offer

  • Interactive JavaScript quizzes covering variables, functions, loops, and arrays
  • Clear code examples with explanations
  • Hands-on practice with a simple calculator tool
  • Progress tracking to monitor your learning journey
  • A clean, beginner-friendly interface

Get Started

Ready to begin your JavaScript journey? Start with our interactive quiz or explore our code examples to see JavaScript in action!
